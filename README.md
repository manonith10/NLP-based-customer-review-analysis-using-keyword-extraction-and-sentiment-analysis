# NLP-based-customer-review-analysis-using-keyword-extraction-and-sentiment-analysis
Extract frequent keywords from feedbac

## Objectives
- Perform text preprocessing
- Extract important keywords
- Analyze customer sentiment
- Visualize customer feedback
- Identify common customer concerns

## Technologies Used
- Python
- Pandas
- NLTK
- TextBlob
- Scikit-learn
- Matplotlib
- Plotly
- WordCloud

## NLP Techniques Used
- Tokenization
- Stopword Removal
- Lemmatization
- TF-IDF Analysis
- Bigram Analysis
- Sentiment Analysis

## Visualizations
- Keyword Frequency Bar Chart
- Word Cloud
- Sentiment Pie Chart
- Bigram Analysis Chart
- Negative Feedback Analysis

## Project Outcome
The project successfully identified major customer concerns and analyzed overall customer sentiment using NLP techniques.
